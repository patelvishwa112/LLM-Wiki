# I Built a Personal AI Assistant With Claude for $0 - Here's Exactly How

**Author:** @0xMortyx (0xMorty)
**Source:** https://x.com/0xMortyx/status/2063967160620245116
**Date:** 2026-06-08

---

Every "build your own AI assistant" tutorial ends the same way: connect this $30/month tool to that $20/month tool, pay for API credits, wire it all together in some automation platform. By the time you're done you're paying $80+ a month and maintaining plumbing that breaks every other week.

I built mine differently. Everything lives inside Claude. No external tools, no automation platforms, no API costs. The only thing I pay for is the Claude subscription I already had. Net new spend: $0.

## The Build

### 1. Home Base — One Project
Your assistant needs a permanent home — a Claude Project. It holds a system prompt and context files. Every conversation inside it starts with full context. Sidebar → Projects → New Project → "My Assistant".

### 2. The Brain — Master Prompt
Custom instructions defining role, communication style, format, and boundaries. Modes include: "brief me", "plan my day", "research [X]", "build me [X]". One paste = assistant has personality and a job.

### 3. The Identity — Memory + Context Files
Three text files uploaded to the Project: about_me.txt, preferences.txt, goals.txt. Combined with Claude's Memory, the assistant gets sharper every week — no re-explaining, ever.

### 4. The Senses — Web Search
Turn on web search for live research: current news, prices, fact-checking. Makes "brief me on AI today" a real command instead of stale training data.

### 5. The Hands — Artifacts
Assistant builds functional tools on demand: habit tracker, daily planner, revenue calculator, decision matrix. Each takes one sentence to request.

## Daily Usage
- Morning: "brief me" — searches niche, checks goals, delivers what matters in 3 minutes
- Planning: "plan my day" — top 3 priorities from goals.txt
- Working: "research [X]" or "draft [Y] in my voice"
- On demand: "build me [tool]" — working artifact in under a minute
- Evening: "what did I not get to, and what should move to tomorrow?"

The unlock: one Project with persistent context means the assistant on day 30 is dramatically better than day 1 — context compounds.

## Limitations
- No autonomous scheduling (manual trigger is intentional — user stays in the loop)
- No cross-app actions without connectors
- Usage limits on heavy days
- 90% of what people pay $80/month in stacked tools for can be done inside Claude with zero extra spend. Start there, add paid plumbing only when needed.
