A non-technical guide to becoming an AI power user in 6 levels, written in deadpan IKEA-instruction style.

## Level 1: Claude.ai basics

- Go to claude.ai, create account, verify phone
- Pick Opus 4.8 (paid). Set thinking level to High. Turn on "Thinking"
- Don't learn to prompt "like an engineer"
- **The trick:** use AskUserQuestion — Claude asks YOU questions instead of you prompting it badly
- Prompt: "Help me do [X] for [Y]. Use AskUserQuestion first."
- Claude asks 3-5 questions. You click answers or type in the "Something else" box
- For pro mode: "give me 3 different strategies"
- That one line — "ask me questions first" — makes you a power user

## Level 2: Talk to your computer

- You talk 4x faster than you type
- Use Wispr Flow (free) — voice dictation, AI-powered
- Talk → computer writes

## Level 3: Claude Cowork (desktop app)

- Download Claude desktop app
- Click the "Cowork" tab (next to Chat)
- Cowork makes actual files saved on your computer — real PDFs, real spreadsheets
- Create a folder called "cowork" on your desktop
- Inside Cowork, select that folder as the project directory
- Drag in messy docs, old proposals, numbers — Claude reads everything
- Use the trick: "build a spreadsheet to do [X]. but ask me what you need first."
- 3 minutes later: clean Excel file on your computer

## Level 4: Connect your apps

- Left sidebar → Customize → Connectors
- Connect Gmail, Calendar, meeting notes (Granola, Otter, etc.)
- Say yes to everything
- Test: prompt something that requires context across all three
- AI just needs context. It needs to know you.

## Level 5: Train Claude with Skills

- Left sidebar → Customize → Skills
- Skills are premade playbooks Claude runs when the task fits — like an employee who knows standard procedures
- Type `/skill-creator` — Claude made a skill to create skills
- Say: "I want to teach Claude how to make Excel spreadsheets the way I love them. But ask me questions first."
- Answer the questions → Claude builds the skill → Save
- Now you can type `/excel-style` in any prompt
- Skills can be shared with your team

## Level 6: Claude Code (coding without knowing how to code)

- Open Claude Code (the `</>` tab in left sidebar)
- Turn on "Bypass permissions" (means "go. i trust you.")
- Same trick: "build a page where people book a 30-min call with me. ask me questions first."
- AskUserQuestion pops: dark mode? bookings to Gmail or Calendly? Click.
- 4 minutes later: "done. open this." It works.
- Want it on mobile? "make this an app i can put on my phone. ask me first."

## The Core Insight

The bar to knowing is trying. Most people say "I'm so behind" but never even tried. Being good at AI is stupidly simple: ask Claude to ask you questions, then click the answers. The author dropped out of university, can't code, now builds things for enterprises.
