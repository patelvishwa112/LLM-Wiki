# OpenThoughts-Agent: Data Recipes for Agentic Models

Sergio Paniego (@SergioPaniego) — announcement thread (quotes Richard Zhuang / Open Thoughts release)

**Problem:** Agents are everywhere; little is public on how to curate training data for agentic models.

**Approach:** Fully open 6-stage data pipeline, tuned with 100+ ablations:

`source tasks → mix → filter tasks → generate rollouts → filter rollouts → pick the teacher`

## Findings

- Task selection matters most (up to ~30 points); diversity beats more rollouts on the same task
- Strongest model ≠ best teacher — GPT-5.3-Codex most capable but weakest teacher; **GLM-4.7** wins as teacher
- Longer agent traces (5+ turns) train better
- Filtering to harder tasks adds a few points
- Rewriting/hardening tasks mostly backfires — untouched tasks win

## Result

**OpenThinkerAgent-32B** — claimed SOTA open-data model at ≤32B: **44.8% avg** across 7 benchmarks; strong generalist; SOTA on 4/5 held-out benchmarks never trained on.

## RL angle (8B)

- `pymethods2test` wins (Codeforces-style Python contracts: docstrings + unit tests)
- Moderately trained SFT model gains more from RL than a strong distilled one
- SFT + RL beats either alone
