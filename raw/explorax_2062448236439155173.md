# 50 MCP Servers That Give Claude, Codex & Gemini Superpowers

The MCP (Model Context Protocol) started as a niche Anthropic spec in late 2024. Eighteen months later it's the standard plug every serious AI agent speaks — Claude Code, Gemini CLI, Codex, Cursor, Windsurf. Over 20,000 servers sit on public registries and the SDK was pulling ~97 million downloads a month by March 2026. Most are junk. A recent audit found security issues in roughly two-thirds of popular ones.

**Three rules that matter more than the whole list:**
1. Don't install 50. More than 5-7 connected servers and your agent starts choking on tool bloat — it gets slower and dumber. Pick the 3-5 that match what you actually do.
2. Treat every server like a CLI from a stranger's GitHub. Pin versions. Scope tokens to read-only until you've watched how the agent uses them. Prefer vendor-official over random forks.
3. Never enable writes against production from an agent loop. Point at a read replica or a snapshot.

## The Universal Core — Install These First

**1. GitHub** — repos, PRs, issues, code search. Official server replaced all community forks in 2026.
```
claude mcp add --transport http github https://api.githubcopilot.com/mcp/
```
Scope the token read-only unless the agent genuinely needs to push.

**2. Context7** — stops agent hallucinating package APIs by pulling current, version-pinned docs at query time.
```
claude mcp add --transport http context7 https://mcp.context7.com/mcp
```
Free, optional API key for higher rate limits.

**3. Playwright** — real browser the agent can drive. Microsoft maintains it.
```
claude mcp add playwright npx @playwright/mcp@latest
```
Great for exploratory testing, wrong for release-critical regression.

**4. Filesystem** — scoped local file access beyond the agent's working directory.
```
claude mcp add filesystem npx @modelcontextprotocol/server-filesystem ~/projects
```
Only expose directories you want reachable. Treat like a broad permission.

**5. Brave Search** — fast web search inside the agent.
```
claude mcp add brave-search npx @brave/brave-search-mcp-server
```
Needs a free key. The old reference server was archived.

## How to Actually Choose

- Don't install 50. Pick the 3-5 that match what you do. This is a menu, not a shopping spree.
- The universal core (GitHub, Context7, Playwright, Filesystem, Brave Search) earns its slot on almost any machine.
- For databases, prefer your provider's official server over the archived reference ones.
- Start everything in read-only. Flip writes on once you trust the loop.
