I tried 100+ Claude Skills, These 15 are the ones that stuck!

Most people treat Claude like a vending machine. type a request, get an output, repeat the whole explanation tomorrow.

Skills flip that. you install someone's expertise once, and Claude starts working your way without you re-explaining anything.

So I went deep. Reddit threads, newsletters, GitHub repos with tens of thousands of stars. I filtered out the hype (looking at you, Caveman) and kept testing until I had the 15 that actually earn a spot in my daily work.

One rule for this list: every single skill is on GitHub. No "sign up for my newsletter to get the file," no dead links. A repo you can open, read, and install in under a minute.

Building with claude code

1. Superpowers
Repo: https://github.com/obra/superpowers
Install: /plugin marketplace add obra/superpowers-marketplace then /plugin install superpowers@superpowers-marketplace

Forces Claude to work like a senior dev instead of an eager intern. Plan first. Work in an isolated environment. Write tests before code. Review its own work twice — once for "does this match what I asked" and once for quality. Gets your first pass to 80% instead of 60%.

2. GSD (Get-Shit-Done)
Repo: https://github.com/gsd-build/get-shit-done
Install: npx get-shit-done-cc@latest --claude --local, then /gsd:map-codebase

Superpowers' bigger sibling. Runs builds in phases. Reads existing code to pick up mid-flight. Slower and more methodical, with safety checks baked in. On big builds, planning time is debugging time saved.

3. Context Mode
Repo: https://github.com/mksglu/claude-context-mode
Install: Run plugin install commands from README, restart Claude Code. Verify with /context-mode:ctx-stats.

Routes big tool outputs through a sandbox so only relevant slices enter the context window. 315KB of raw output becomes ~5KB over a session. Sessions that died at 30 minutes run for hours.

4. frontend-design (official)
Repo: https://github.com/anthropics/skills/tree/main/skills/frontend-design
Install: Ships with paid Claude by default.

Anthropic's own skill for HTML/CSS/React with Tailwind and shadcn, with real attention to spacing, alignment, and hierarchy. Good design is mostly restraint and consistency — a checklist enforces both better than vibes.

Getting actual work done

5. content-research-writer
Repo: https://github.com/ComposioHQ/awesome-claude-skills/tree/master/content-research-writer
Install: Drop folder into ~/.claude/skills/

Acts as a research-and-writing partner: hunts references, adds citations, sharpens hooks, iterates on outlines, gives section-by-section feedback while holding your style. Splits the job like good writers do — research first, voice second, line-edit last.

6. document-skills (Docx, Xlsx, Pptx, PDF — official)
Repo: https://github.com/anthropics/skills
Install: Built into paid Claude by default.

Full suite for creating and editing real Office files. Models in Excel, formatted reports in Word, decks in PowerPoint, filled forms in PDF. Encodes the trial-and-error of producing professional documents that don't look robot-exported.

7. Meeting Insights Analyzer
Repo: https://github.com/ComposioHQ/awesome-claude-skills/tree/master/meeting-insights-analyzer
Install: Drop folder into ~/.claude/skills/

Analyzes meeting transcripts for behavioral patterns, conflict avoidance, speaking ratios, filler words, facilitation style. Goes past "here are the action items" into "here's how you communicated."

8. NotebookLM Integration
Repo: https://github.com/ComposioHQ/awesome-claude-skills (notebooklm entry)
Install: Follow folder's SKILL.md

Lets Claude Code chat directly with NotebookLM for source-grounded answers. NotebookLM has the grounding, Claude has the automation — source-checking without tab-switching.

Thinking, research and reach

9. interview-me
Repo: https://github.com/Sorbh/interview-me
Install: curl the SKILL.md into ~/.claude/skills/interview-me/

Reads requirements, scans codebase, asks hard questions one at a time, pushes back on contradictions, probes edge cases, then produces a production-grade spec with decisions log. When you're being interrogated by a senior-architect persona, you get pushed out of your existing frame — where the gaps live.

10. Twitter Algorithm Optimizer
Repo: https://github.com/ComposioHQ/awesome-claude-skills/tree/master/twitter-algorithm-optimizer
Install: Drop folder into ~/.claude/skills/

Analyzes and rewrites tweets using insights from Twitter's open-sourced ranking algorithm, optimizing against how the recommendation system actually ranks content.

11. academic-research-skills
Repo: https://github.com/imbad0202/academic-research-skills
Install: Follow docs/SETUP.md

Full research workflow: research → write → review → revise → finalize. Hunts references, formats citations, verifies data, checks logical consistency, calibrates to writing voice. Helps you write better, not hide AI.

The meta tier (skills about skills)

12. skill-creator
Repo: https://github.com/anthropics/skills/tree/main/skills/skill-creator
Install: Built into paid Claude.

Generates standardized skill files and folder structures from plain-English descriptions. The hardest part of skills isn't the idea, it's the scaffolding and silent formatting errors — this removes both.

13. one-skill-to-rule-them-all
Repo: https://github.com/rebelytics/one-skill-to-rule-them-all
Install: Clone repo, add load instruction to CLAUDE.md, schedule recurring review task.

Meta-skill that observes work sessions, logs improvement opportunities for all skills (including itself), and applies them in scheduled autonomous sessions. Creator logged 600+ improvements across ~40 skills in three months.

14. skill-optimizer
Repo: https://github.com/BehiSecc/awesome-claude-skills (skill-optimizer entry)
Install: Drop folder into ~/.claude/skills/

Diagnoses and improves existing skills using session data and static analysis. Flags what to tighten, merge, or cut. Treats your skill library like code that needs maintenance.

15. awesome-claude-skills (the directory)
Repo: https://github.com/ComposioHQ/awesome-claude-skills

Curated, categorized index of hundreds of Claude skills organized by use case. Also worth checking: github.com/travisvn/awesome-claude-skills and github.com/VoltAgent/awesome-agent-skills.

Skip the hype: Caveman was debunked — it only trims tokens on final output, not the thinking.

How to install a GitHub skill (30-second version)
- Claude Code: Drop skill folder into ~/.claude/skills/, restart Claude Code. Plugin-style repos use their own /plugin or npx install command.
- Claude.ai / Cowork: Download .zip or SKILL.md → Customize → Skills → "+" → Upload.

Safety: paste the repo link into Claude and ask it to check for anything malicious before running unfamiliar skills.

Bottom line: Start with 3 — Superpowers, interview-me, and skill-creator. The gap between people who use skills and people who don't is going to get embarrassing fast.
