## What Cowork Actually Does

- File processing: folder of PDFs → extract + summary spreadsheet. 3 hours manual → 5 minutes.
- Document creation: meeting notes → project proposal using templates.
- Data organization: sort Downloads by type, move files, delete old.
- Connected workflows: Gmail, Calendar, Drive, Slack, Notion — read, draft, create, post.
- Scheduled automation: daily briefing, weekly cleanup, monthly reports — run automatically.
- Dispatch mode: control desktop automation from phone.

"This is not a chatbot. This is a digital employee that works on your actual computer with your actual files."

## The 14-Day Onboarding Plan

### Day 1: Setup
Download Claude Desktop, open Cowork tab. Grant access to Documents, Downloads, Desktop. First task: "Read all files in my Downloads folder. Create a summary of what each file is and when downloaded. Organize into a table sorted by date."

### Day 2-3: Connect Services
Connect Gmail (read inbox, draft responses), Calendar (schedule, conflicts, open slots), Drive (read/write docs), Slack (read/post/search), Notion (pages, databases). Each takes 2-3 clicks.

### Day 4-5: Morning Routine
Scheduled 7:30am task: check overnight emails (top 5 by importance), today's calendar, Slack DMs/mentions, yesterday's project notes → compile into Daily Briefing doc. Claude: 3 minutes. Manual: 20-30 minutes.

### Day 6-7: File Processing System
- Invoice Processor: extract vendor, number, amount, due date, line items → spreadsheet. Flag due-within-7-days.
- Receipt Organizer: identify store, date, total, category from photos → monthly expense summary.
- Report Compiler: combine last 5 team updates into weekly summary.

### Day 8-10: End of Day System
Evening close-out: scan today's created/modified files, Slack thread outcomes, sent emails awaiting responses → End of Day note with accomplishments, pending, tomorrow's priorities.

### Day 11-14: Domain-Specific Automations
- Marketing: weekly industry news search → trend summary
- Sales: categorize leads by stage → pipeline summary
- Operations: expense report analysis → category summary with anomaly flags
- Student: lecture notes → study flashcards
- Team manager: project updates → status report (on track/at risk/blockers)

## The Power of Dispatch

Trigger automations from phone while away. "Process three documents in /Contracts folder. Summarize key terms. Flag unusual items. Save to Drive. Slack me when done." You're in a meeting. Claude works on your laptop. Slack notification when done.

## Multi-Step Workflows

**Client Report Pipeline (6 steps, 3-5 min vs 45-60 min manual):**
1. Read project notes
2. Check Slack for client mentions
3. Check Drive for updated shared docs
4. Compile formatted status report
5. Save to Drive + draft email with attachment
6. Post summary to client Slack channel

**Meeting Prep Automator:** Every evening, for each tomorrow meeting: find last meeting with attendees, pull relevant notes, search Gmail threads, create one-page prep doc.

**Content Batch Processor:** URLs in a text file → summaries with key argument, 3 insights, actionable takeaway, relevance rating → single document.
