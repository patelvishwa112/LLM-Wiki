# Math behind Attention - Q, K, and V

Amit Shekhar (@amitiitbhu) — Apr 3, 2026

**Formula:** `Attention(Q, K, V) = softmax(Q × K^T / √d_k) × V`

**Example sentence:** "I love AI" — 3 tokens, d_emb=4, d_k=3 (toy weights W_Q, W_K, W_V).

## Pipeline

1. Embeddings X (3×4)
2. Q = X W_Q, K = X W_K, V = X W_V (learned in real models)
3. Scores = Q K^T — e.g. "I"→"love" raw score 5 (highest for "I" row)
4. Scale ÷ √d_k (÷1.732) — prevents softmax saturation
5. Softmax per row → attention weights (sum to 1)
   - "I" row: 7.0% self, **70.7% love**, 22.3% AI
6. Output = weights × V — contextualized vectors per token

## Takeaway

Each token output is a **weighted mix of all Value vectors**; weights from Query–Key compatibility. Same core op as production LLMs at larger d.
