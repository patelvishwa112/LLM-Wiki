## Post (10 items)

### 1. Red Queen Gödel Machine
Self-improving agents plateau when the evaluator is frozen. Framework co-evolves agents and utility/evaluator at epoch boundaries (Red Queen race).

### 2. MCP Server Patterns
Industry experience paper: five patterns from 15 servers — Resource Gateway, Tool Orchestrator, Stateful Session Server, Proxy Aggregator, Domain-Specific Adapter; anti-patterns; auth, versioning, observability.

### 3. The Verification Horizon (Qwen)
Fixed rewards get gamed as policies strengthen; verifiers must co-evolve. Four reward constructions; axes: scalability, faithfulness, robustness.

### 4. Paper Assistant Tool (Google)
Agentic deep peer review — theory, experiments, flaws; ladder from author tool → AI reviewer; AIrXiv-style automated review vision.

### 5. Generative Skill Composition (SkillComposer)
Joint generation of which skills, how many, in what order (not independent retrieval). SkillsBench gains at lower token cost.

### 6. AutoMem (Stanford)
Metamemory: read/write/search/append in action space; dual meta-learning loops; 2–4× on Crafter, MiniHack, NetHack.

### 7. RLMF (Google + Yale)
Metacognition as RLHF signal for calibration; two-stage faithfulness + linguistic uncertainty.

### 8. ASPIRE
Robot code-as-policy continual learning; skill library + evolutionary search; +77% on perturbed manipulation.

### 9. HORIZON
Hardware design as git repo evolution; ChipBench, RTLLM, Verilog-Eval, CVDP hands-free completion.

### 10. Reasoning Quality Emerges Early (UCLA)
Prefix predicts trace quality; difficulty from first ~100 tokens at perturbed checkpoint — cheap SFT curation.
