## Content

My apologies to DeepSeek-V4 Pro. I was NOT AWARE about your Three.js game.

These buildings are 4000 responses to tasks by a small language model I'm training. Similar responses are close to each other. Color coded by task type, height is determined by generation quality/correctness.

It started coz I wanted a cool visual for my upcoming youtube video.

I generated thousands of responses at different temperature settings -> ran distill-bert embedding -> umap+pca to convert into 2D space -> use a judge LM (local Qwen) to evaluate correctness and coherence + Vendi scores to calculate semantic diversity + EAD for lexical diversity.

All I needed was a cool way to display all that for the Youtube audience. DeepSeek just got so much of it done - (yes it took handholding) - from running embedding models, to writing the judge script using outlines, and then building this interactive page.
