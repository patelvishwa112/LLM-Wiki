# How to use Fable to orchestrate a huge project with 40 sub-agents

**Author:** Ryan Carson (@ryancarson), Hello Untangle  
**Posted:** Jul 6, 2026  
**Stack:** One Fable parent + ~40 Devin Ultra child sessions

## Program outcome

834 files, prod data mutation, DB schema update, 31 PRs, Fri→Mon, **zero prod incidents**. Parent orchestrator planned, spawned children, enforced regression/backup gates, escalated scope rulings and irreversible go/no-go only.

## Architecture

Parent: plan, spawn, review, sequence phases, escalate — **writes zero code**. ~40 children: 11 parallel Phase 0 audits, per-wave execution, dedicated gate-runner sessions. Avoids **context collapse** on long programs.

## Seven patterns

1. **Audit first + frozen manifest** — 11 parallel audits → one committed manifest (verdict + wave per unit). No re-litigation; UNKNOWN → stop, owner ruling recorded in manifest.
2. **Risk-ordered waves** — 0 audit/tag → 1 no dependents → 2 entanglement surgery → 3–4 bulk code → 5 prod data (dry-run inventory) → 6 schema → 7 deps/docs/knowledge. Irreversible steps last.
3. **Disjoint file boundaries** — parallel children get explicit non-overlapping paths; no merge conflicts.
4. **Safety gates between waves** — E2E regression (fresh DB branch), backup audit + PITR snapshot before prod, dry-run row counts + post-verify, git restore tags; failed gates → scoped fix sessions.
5. **Humans approve; agents propose** — destructive work: proposed inventory → human approves exact list → execute only that list.
6. **Escalation protocol** — children stop/report to parent on manifest mismatch; parent updates manifest (demotions, audit reversals).
7. **Leave campsite cleaner** — Wave 7: AGENTS.md, repo skills, agent knowledge base updated for new architecture.

## Cost (metered)

40 sessions total ~**$1,155** (parent $115; Phase 0 audits+manifest ~$86; entanglement $299; regression gates $245; prod data step **$15** after risk engineered out). Note: cheaper if child sessions could use lighter models (gpt-5.5/opus-4.8) per task.

## Takeaway

Program management delegated: decomposition, sequencing, parallelization, gating, escalation inside orchestrator; human = owner-only decisions.
