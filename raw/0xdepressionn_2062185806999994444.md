# 15 Prompts That Cut My Coding Bill from $62,000 to $7,800/month

A team was spending $62,000/month on Claude Opus alone. Then they discovered Opus 4.8's Dynamic Workflows were designed for orchestration, not execution. They moved execution to Kimi K2.6 Agent Swarm. Same quality. $7,800/month. Saving $650,400/year.

**The core insight:** Opus 4.8 is a strategist (plan, reason, judge, synthesize). Kimi K2.6 is an executor (300 parallel sub-agents, 4,000 coordinated steps, $0.60/M tokens). They're not competing — they're different layers of the same system.

**The routing rule:** If you can write a rubric a machine could grade, Kimi executes. If you can't, Opus handles it.

## Section 1: Orchestration (Prompts 1-5)

**Prompt 1 — Project Planner:** "I need to execute the following project using multiple AI agents. Your job is to build the execution plan, not execute it."

**Prompt 2 — Dynamic Workflow Task Routing:** Sort tasks into Group A (Opus handles: reasoning-heavy, judgment, high-stakes) and Group B (Kimi handles: execution-heavy, batch, parallel-friendly).

**Prompt 3 — Quality Standard Definition:** Define rubric: what passing/failing looks like, 3 most common failure modes, how to catch each before final assembly.

**Prompt 4 — Opus Reviews Kimi Output:** Review against rubric. Pass/fail per criterion. Quote exact problems. Produce revision brief directly sendable to Kimi.

**Prompt 5 — Opus Assembles Final Output:** Connect and integrate Kimi's components. Fix inconsistencies. Flag anything requiring human review.

## Section 2: Execution (Prompts 6-10)

**Prompt 6 — Translate Opus Plan to Kimi Brief:** Rewrite Opus plan as Kimi Agent Swarm brief with input/output/phase specs.

**Prompt 7 — Batch Execution with Output Spec:** Run N inputs in parallel. Exact format, file type, naming convention, quality standard.

**Prompt 8 — Research-to-Deliverable in One Pass:** Research phase → Analysis phase → Deliverable phase. One structured output.

**Prompt 9 — Save Workflow as Reusable Skill:** Document trigger, Opus orchestration prompt, Kimi execution template, Opus review checklist.

**Prompt 10 — Cost Tracking Per Run:** Calculate Opus vs Kimi vs all-Opus cost. Log to WORKFLOW_COSTS.md.

## Section 3: The System (Prompts 11-15)

**Prompt 11 — Weekly Workflow Audit:** Classify every task Opus-only/Kimi-only/Hybrid. Flag Opus tasks that should move to Kimi.

**Prompt 12 — Build Routing Decision Tree:** For each workflow type, define model+trigger+handoff format+cost estimate.

**Prompt 13 — Monthly Cost Optimization:** Analyze over-allocation, optimal split, projected savings with routing changes.

**Prompt 14 — Handoff Template:** Create permanent Opus→Kimi brief template + Kimi→Opus review checklist + revision loop format.

**Prompt 15 — ROI Report for Stakeholders:** One-page executive summary: workflows run, volume, cost breakdown, all-Opus equivalent, monthly/annual saving, quality incident rate.

## The Numbers

- Opus orchestration (30%): $18,600/month
- Kimi execution (70%): $1,240/month
- Total: ~$19,840/month vs $62,000 all-Opus
- Annual saving: $650,400
