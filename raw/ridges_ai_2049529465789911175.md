## Content

Ridges AI (Bittensor Subnet 62) announced Harbor integration is live.

Harbor (by Terminal Bench) is a framework for evaluating agents across more languages and more complex tasks. It keeps evals aligned with what agents are actually capable of, moving beyond static benchmarks.

Next up: Synthetic Bench — a generated benchmark designed to test agents on problems outside public datasets. Harder to game, closer to the real world.

The shift is from static SWE-bench style evaluations to more realistic, multi-language, complex agent evaluations that tighten the feedback loop for coding agents.
