## Core insight

If you have to remind an agent how to work more than twice, that behavior probably belongs in a **skill**. Turn "please do this every time" into environment that auto-invokes.

Start with boring, high-leverage habits: design before implementation, identity before autonomy, repo memory before rediscovery, session memory before context resets, safe workspace access before letting the model click around, diagrams before hand-wavy systems, skill creation before repeating workflows forever.

## Seven skills for a serious Hermes agent

1. **Superpowers (Brainstorming)** — Design before build: understand project, one clarifying question at a time, propose approaches, get approval, then plan. Prevents building the wrong thing confidently. Use for complex tasks with tradeoffs, workflows, specs, systems.

2. **Soul-Grader-Skill** — Grades/enforces strong `soul.md` (compact constitution): mission, boundaries, authority, truthfulness; prevents overstepping and false claims. Use when creating/revising profiles or agent feels generic.

3. **Project-Memory-Skill** — Append-only `docs/memory/...` in repo: decisions, learnings, gotchas, next steps. Memory travels with the repo.

4. **Agent-Memory-Wiki** — Obsidian-based memory: daily/session logs, safe vault access, janitor cleanup. Curated human-readable notes vs raw transcript dumps.

5. **gogcli** (Google Workspace CLI) — Recommended adjunct: Gmail, Calendar, Drive, Docs, Sheets with allowlists, dry-run, audit. Pair with a skill for approval gates.

6. **Hermeshub Diagram Maker** — Valid Mermaid from natural language; catches bad syntax, oversized graphs, reserved words. "If it can't diagram it, it doesn't understand it."

7. **Hermeshub Skill Factory (meta)** — Watches repeated workflows, mistakes, tool combos, corrections → new `skill.md` files.

## Bottom line

A serious Hermes agent needs **working habits** encoded as skills — how to start, remember, visualize, and improve. Watch failure modes; add the skill that fixes them; stop re-explaining the same procedure every session.
