# Claude Can Do All of This. Most People Have No Idea.

By Anatoli Kopadze (@AnatoliKopadze) | May 22, 2026 | 2.7M views | 4,950 bookmarks

Complete guide to every Claude feature — where to find it, how to turn it on, and a prompt you can use immediately.

## Hidden Features in Regular Claude

### 1. Projects — Claude that remembers you
Create a project, upload documents, write standing instructions. Claude holds all permanently. Best prompt: role definition with tone, format, and content rules.

### 2. Artifacts — Working apps inside chat
Live products in a side panel — calculators, habit trackers, games, dashboards. SVG, interactive charts, Mermaid diagrams. Free plan. Example: "Build me a habit tracker as a working web app with 5 daily habits, 7-day streak counter, dark background, satisfying click animations, persistent data."

### 3. Adaptive Thinking — Extended reasoning
Claude reasons step by step before answering. For complex decisions where you want actual thinking, not pattern-matching. Example prompt: decision analysis with second/third-order consequences, emotional biases, missing information, downside protection.

### 4. Memory — Claude that knows who you are
Builds a profile over time. New chats already know your context. Off by default.

## Role Prompts

### 5. Personal psychologist — CBT therapist
Questions instead of answers. Challenges thinking instead of validating. Identifies cognitive distortions.

### 6. The hard mentor — Brutally honest
Disagrees specifically. Points out what you're avoiding. Asks hard questions. Ends with the single most important thing to think about.

### 7. Personal trainer — Custom 12-week program
Accounts for age, goals, equipment, schedule, injuries. Adjusts weekly based on feedback.

### 8. Practice difficult conversations — Roleplay
Claude plays the other person realistically. Pushes back on weak arguments. Steps out of character for feedback between exchanges.

### 9. Devil's advocate — Case against your decision
Builds the strongest possible counterargument. Finds assumptions that could be wrong. 3 most realistic failure modes.

## Product Features

### 10. Claude in Chrome — Sees what you see
Browser extension. Reads page, clicks links, fills forms, navigates. Extract data from job listings across pages.

### 11. Claude Cowork — Desktop app
Direct file system access. Reads, edits, creates files. No copy-paste.

### 12. Scheduled Tasks — Works while you sleep
Auto-execute at chosen times. Example: daily 7:30am AI/crypto news brief saved to /briefs folder.

### 13. Skills in Cowork — Installable capabilities
Pre-built instruction sets. Like apps on a phone. PowerPoint, PDFs, specific workflows.

### 14. CLAUDE.md — Auto-read rules
Project-level conventions. Coding standards, writing rules, terminology. Read at start of every session.

### 15. Claude Code — Terminal coding agent
Reads codebase, writes code, runs tests, fixes bugs in a loop. VS Code, JetBrains, GitHub Actions.

### 16. Claude Design — Visual work
Product one-pagers, pitch decks, prototypes, landing pages. Exports to PPTX, Canva, PDF, HTML. claude.ai/design.

### 17. Prompt Caching — 90% cost reduction (API)
Cache repeated context blocks server-side. Subsequent calls reuse cache. Add `cache_control: {type: "ephemeral"}` to content blocks.
