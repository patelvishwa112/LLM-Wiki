## Post

The AI workflow in this article is ridiculously underrated.

You know the feeling of explaining yourself over and over to different AIs. Claude.MD files are a solid solution for most cases, but what if there is a better way? The author built a simple AI second brain (level 2 after Claude.MD), and a system that can THINK—not just remember.

## How does the AI second brain work?

The system connected all notes using Karpathy's wiki method: reads everything ever made and compiles it into a living wiki, one page per topic.

Every time an agent runs, it writes back into the wiki first, so the brain gets smarter every single time. New topic? It opens a new page on its own. The wiki never stops growing.

Hermes agents go through wiki connections plus prior data (238 articles, 1,391 Claude.ai conversations, 138 Claude Code sessions) and generate a thought that can change everything.

Connections (four agents):

- **Post** — Reads old articles + builds, finds what actually made money (with live proof from the web)
- **Build** — Looks at what was built but never shipped, then tells what to make next
- **Stoic** — Reads journal + psychology profile, coaches through the day
- **Note** — Studies best Substack notes and hands the next one to write

Every six hours they update the wiki and write something new to the journal while the user sleeps.

Examples: Build agent used Claude Code conversation history, created a new SEO page, referred to a prior $50K website-clone project; cross-checked article data and X post stats for conversion potential. Stoic agent used a Leo-Psychology page built from Claude.ai conversation analysis plus daily journal.

## Setup (summary)

1. **Obsidian** — new vault folder is the brain
2. **Export Claude.ai** — Settings → Privacy → Export data (zip)
3. **Process conversations** — strip noise to clean notes; build psychology profile; filter private chats; same for `~/.claude/projects/` sessions (builds never shipped flagged)
4. **SKILL.md + SOUL.md** — Claude suggests wiki connections; user approves; wiki forms page by page
5. **Automation** — Option 1: pass SOUL.md to Hermes agent, give vault path, cron every few hours. Option 2: local Claude routine when PC is on
