Claude Cowork is Anthropic's knowledge work agent. Running in the Claude desktop app, it reads and writes local files, works across connected apps like Slack and Google Drive, and carries multi-step tasks through to real deliverables with citations back to actual files and messages.

**What makes Claude Cowork different:**
- Local file access, subagents, long-running work, and scheduled tasks
- Cross-app knowledge work (not just conversational)
- You describe the goal, desired outcome, and cadence — Claude lays out the steps and does the work

**Product matrix:**
- Chat → conversational drafting
- Claude Code → coding
- Claude Cowork → cross-app knowledge work

**Seven common workflows:** research briefs, meeting prep, recurring reports, and more.

**Plugins:** available for specific knowledge work like marketing and product management.
