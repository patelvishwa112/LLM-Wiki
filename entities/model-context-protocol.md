# Model Context Protocol (MCP)

An open standard, introduced by Anthropic in late 2024, for connecting LLM
assistants to the systems where their data and tools live — IDEs, business
software, content stores, dev environments. MCP is the substrate underneath
Claude Desktop Extensions and the practical mechanism that lets [[ai-agents]]
plug into arbitrary tooling.
