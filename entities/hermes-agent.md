# Hermes Agent

Personal AI agent platform by Nous Research. Persistent memory, scheduling, multi-platform messaging, skill system, and Kanban orchestration.

## Key Facts

- Open-source agent harness
- Skill system with extensions and presets
- /goal mode for autonomous execution
- Signal, Discord, Telegram integrations
- Cron scheduler for recurring tasks
