# Computer Use

Computer Use is the capability for Claude to operate a desktop environment
directly — taking screenshots, moving the cursor, clicking, and typing —
turning the model into an agent that can drive arbitrary GUI software.
Introduced with Claude 3.5 Sonnet in late 2024, it is the bridge between
Claude as a chat model and Claude as an autonomous [[ai-agents|agent]].
