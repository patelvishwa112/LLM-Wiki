# AI Safety Levels (ASL)

ASL is Anthropic's tiered classification of model risk, modelled on biosafety
levels. Each ASL prescribes the security posture, deployment safeguards, and
evaluation rigor required to develop and release a model at that capability
tier. ASLs are the rungs of the [[responsible-scaling-policy]] ladder.

## What the sources say

- [[anthropic-news-activating-asl3-protections]] — Marks the operational
  transition: Anthropic activated ASL-3 protections (enhanced security and
  deployment safeguards) for a Claude model release
- [[responsible-scaling-policy]] — Defines the ASL ladder
- [[anthropic-news-responsible-scaling-policy-v3]] — V3 of the policy
  refining ASL thresholds

## Key dimensions per ASL

- Capability evaluations that trigger the level
- Information-security requirements (e.g., weight protection)
- Deployment safeguards (classifiers, monitoring)
- Pre-deployment evaluation requirements
