# LLMs-local: Curated List of Local LLM Platforms, Tools, and Resources

**Source:** [0xSojalSec/LLMs-local on GitHub](https://github.com/0xSojalSec/LLMs-local)

## Summary
A comprehensive, well-organized GitHub repository curating platforms, inference engines, user interfaces, models, tools, hardware recommendations, tutorials, and communities for running Large Language Models locally. Covers the full stack from inference (Ollama, llama.cpp, vLLM, MLX, etc.) to UIs (Open WebUI, SillyTavern), agent frameworks, RAG, computer use, browser automation, evaluation, training/fine-tuning, and specialized hardware (Apple Silicon, AMD, Intel NPUs, distributed setups). Includes links with GitHub star badges for popularity signals and sections for explorers/benchmarks, specific model categories (coding, multimodal, audio), and tutorials on prompt/context engineering and agents.

This is high-signal raw material for the LLM wiki's "ML Research" and "local inference" domains.
