# Hermes Agent Use Cases — Multi-Agent Setup and Philosophy

## Summary
A practical guide to setting up and using Hermes Agent in a multi-agent configuration. The author runs multiple specialized agents with different profiles and models, accessed via Hermes TUI and Telegram. Core philosophy: AI is an assistant for grunt work, not a replacement for thinking.

## Key Takeaways

**Philosophy:**
- Treat AI as an assistant, not a thinking replacement
- AI does grunt work → human verifies and proceeds
- Identify tasks that take time but provide low value

**Discovery Method:**
- Write down everything you do in a day
- Ask: "What took a lot of time?" and "What didn't provide much value?"
- Also consider life friction — things you forget or that make life harder

**Agent Crew Setup:**
- Tech Research Agent with multiple profiles
- Different profiles use different providers/models
- Accessed via Hermes TUI and Telegram

**The Softer Stuff:**
- Day-to-day life issues
- Things you forget to do
- Tasks that add friction to your life
