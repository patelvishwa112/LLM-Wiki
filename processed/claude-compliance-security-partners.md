# Claude Compliance API & Security Partners

**Source:** Anthropic, June 2026. 28 integrations with security and compliance tools, powered by the Claude Compliance API.

## Two Data Types Exposed

| Data Type | Source | What It Covers |
|-----------|--------|----------------|
| **Conversation content** | Claude Enterprise | Chats, uploaded files, projects — apply DLP, monitoring, security policies |
| **Activity events** | Claude Enterprise + Claude Platform | User logins, admin actions, configuration changes — unified usage view |

## Integration Categories & Partners

| Category | Partners |
|----------|----------|
| **DLP / SASE** | Cloudflare, Netskope, Forcepoint, Fortinet |
| **Data Security** | Cyera, Rubrik, Proofpoint, Microsoft Purview |
| **SIEM / Security Operations** | Datadog, Cribl, ReliaQuest |
| **Identity** | Okta, SailPoint |
| **eDiscovery** | Relativity, Mimecast |
| **AI Security Posture** | IBM Guardium, Geordie AI, Palo Alto Networks |
| **AI Observability** | CrowdStrike |

## How It Works

Connect and configure Claude instance → data flows into existing dashboards and alerting workflows. Security teams govern Claude with the same tools and policies used for other workplace applications — no separate monitoring stack needed.

## Getting Started

- Claude customers: connect via Help Center and Compliance API docs
- Security partners: apply to join the integration network
