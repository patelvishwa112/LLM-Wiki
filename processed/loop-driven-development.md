# Loop Driven Development

Nuno Campos's deep essay on the new engineering discipline.

## Core Thesis
The bottleneck shifted from implementation to the oracle problem: what is the right test or answer? LDD = deliberately choosing the feedback source and letting the agent loop against it.

Relevant to Hermes /goal and agent loop design.
