# Claude Can Do All of This

By Anatoli Kopadze | 2.7M views | 4,950 bookmarks

## Key Takeaways

- **17 features, most unused.** Projects, Artifacts, Extended Thinking, Memory, role prompts (CBT therapist, hard mentor, trainer, conversation practice, devil's advocate), Claude in Chrome, Cowork, Scheduled Tasks, Skills, CLAUDE.md, Claude Code, Claude Design, Prompt Caching.
- **Projects are the first thing to set up.** Every new chat starts from zero without them. Upload docs, write standing instructions — Claude holds everything permanently.
- **Artifacts build live apps inside chat.** Not code blocks — working products. Free plan. Habit trackers, games, dashboards, SVG graphics.
- **Extended Thinking is a different level.** For complex decisions. Turn it on, ask the same question, compare the difference.
- **Role prompts change everything.** Give Claude a specific identity and it commits fully — therapist asks questions instead of validating, mentor stress-tests ideas, devil's advocate builds the counterargument.
- **Prompt Caching saves 90% on API costs.** Repeated context blocks (system prompts, reference docs) get cached server-side. 5-minute TTL, resets on use.
