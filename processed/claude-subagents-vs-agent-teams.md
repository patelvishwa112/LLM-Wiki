# Claude Subagents vs. Agent Teams, clearly explained

**Author:** Avi Chawla (@_avichawla)  
**Format:** X post sharing external article (link + preview)

## Core Thesis
When a task feels complex, the default impulse is to reach for multi-agent systems. The better question is whether to use **subagents** (hierarchical delegation within a single agent) or **agent teams** (independent collaborating agents). Subagents are often the simpler, more effective starting point.
