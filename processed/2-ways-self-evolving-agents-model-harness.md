# 2 Ways towards Self-Evolving Agents

**Author:** Zengyi Qin (@qinzytech)  
**Format:** X Article (hand-typed, high-signal)

## Core Thesis
Self-evolving agents can be achieved through two primary paths:

1. **Model approach** — Architecture design + training methods that enable the model itself to improve over time.
2. **Harness approach** — Building external systems (harnesses, memory, verification, orchestration) that allow agents to evolve without changing the underlying model.
