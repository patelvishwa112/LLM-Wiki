# I Built a Personal AI Assistant With Claude for $0 - Here's Exactly How

0xMorty's $0 personal assistant setup using only Claude Projects.

## Core Components
1. **One Project** as the permanent home (system prompt + context files)
2. **Master Prompt** defining role, response style, daily modes ("brief me", "plan my day", etc.)
3. **Three context files**:
   - about_me.txt
   - preferences.txt (voice samples, topics to ignore)
   - goals.txt (quarterly focus + constraints)
4. **Web Search** for live research
5. **Artifacts** for on-demand tool building

## The Unlock
Persistent context in one Project means the assistant gets dramatically better over time without re-explaining anything.

Relevant to personal agent setups and context engineering.
