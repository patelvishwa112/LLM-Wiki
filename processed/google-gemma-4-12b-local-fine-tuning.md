# Google Gemma 4 12B — Local 8GB Fine-Tuning

Akshay shares Google's new Gemma 4 12B multimodal model and plans for local fine-tuning.

## Key Points
- Runs on just 8GB VRAM (text + images + audio).
- Fine-tuning stack: Unsloth (efficient) + Hugging Face transformers.
- Target use case: Chess move prediction (master the game locally).

Relevant to local LLM experimentation, MLX/Unsloth workflows, and on-device model training.
