# Building a 100x Cheaper Trace Judge with Fireworks

**Authors:** Viv Trivedy (@Vtrivedy10, LangChain), Jake Broekhuizen (LangChain) et al.  
**Collaboration:** LangChain Labs × Fireworks AI

## Core Claim
Fine-tuning open models on large-scale trace data can produce trace judges that are ~100× cheaper than proprietary alternatives while maintaining signal quality for LLM/agent observability.

## Key Themes
- **Trace evaluation at scale** — Mining signals from massive agent/LLM trace datasets.
- **Cost reduction via fine-tuning** — Moving from expensive closed models to efficient open models specialized for judging.
- **Relevance to agent harnesses** — Better, cheaper judges enable higher-fidelity automated evaluation, verification loops, and self-improvement in production agent systems.
- **Connection to agent memory & fault tolerance** — Improved trace judges directly support memory quality assessment, error detection, and harness reliability.
