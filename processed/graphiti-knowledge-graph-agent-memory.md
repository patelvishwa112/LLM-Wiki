# Graphiti — Knowledge Graph Memory for Agents

By Avi Chawla | 26K+ GitHub stars | Open source

## Key Takeaways

- **3-layer knowledge graph:** Episode subgraph (raw data + timestamps), Semantic entity subgraph (entities, facts, versioned), Community subgraph (clustered entities with summaries)
- **Temporally-aware:** Continuously evolving memory that tracks when information was learned
- **Beats MemGPT:** 18.5% higher accuracy, 90% lower latency
- **Entity-centric:** Extracts entities and relationships from episodes, replaces outdated info
