# Osiris: Open Source Palantir Alternative

## Summary
Osiris is an open-source intelligence and real-time tracking platform described as an "open source Palantir." It runs a 3D interactive globe with day/night cycles and integrates 20+ live API feeds for tracking aircraft, satellites, CCTV cameras, and environmental events, alongside built-in OSINT tools requiring no additional installations.

## Key Takeaways
- **Real-time tracking**: 10,000+ commercial, military, and private aircraft on a 3D globe; 2,000+ satellites including ISS; 1,400+ worldwide CCTV feeds
- **Environmental monitoring**: Earthquakes, wildfires, nuclear facilities, and severe weather tracking
- **Built-in OSINT tools** (browser-based, no installs):
  - Nmap port scanning from the browser
  - DNS record lookup and enumeration
  - WHOIS domain intelligence
  - SSL/TLS certificate transparency
  - BGP routing and ASN lookup
  - Threat intelligence and IP reputation
- **Interface**: Everything runs on a 3D interactive globe with day/night cycle and SIGINT news aggregator
- **Announced on Reddit** as a free, open-source alternative to Palantir
