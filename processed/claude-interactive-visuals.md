# Claude Interactive Visuals

Claude now generates inline interactive charts, diagrams, and visualizations during conversations — designed for understanding, not permanence.

## Key Points

| Aspect | Detail |
|--------|--------|
| Purpose | Aid understanding during discussion, not create permanent artifacts |
| Behavior | Appear inline, temporary — change or disappear as conversation evolves |
| vs. Artifacts | Artifacts are permanent, sharable, side-panel tools/documents; visuals are ephemeral and conversational |
| Control | Claude decides when a visual helps, or user can request directly ("draw this as a diagram") |
| Interaction | User can ask for adjustments or deeper exploration after Claude creates a visual |
| Availability | All paid plans, on by default (in Claude Cowork as of April 2026) |

## Examples

- Compound interest → interactive curve to explore
- Periodic table → clickable visualization with detail drill-down
- Recipes → formatted with ingredients and steps
- Weather → visual presentation

## Context

Part of broader response improvements including purpose-designed formats for specific topics (recipes, weather) and direct interaction with apps like Figma, Canva, and Slack within conversations.
