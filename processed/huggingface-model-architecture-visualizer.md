# HuggingFace Model Architecture Visualizer — hfviewer.com

## Summary
A new tool (hfviewer.com) that lets you plug in any HuggingFace model URL and explore its architecture graph at any level of granularity. Built by Embedl, a company focused on hardware-optimized ultra-efficient SLMs for on-device AI.

## Key Takeaways

- **hfviewer.com:** Architecture graph visualizer for any HuggingFace model — just plug in the URL
- **Granular exploration:** Explore model architecture at any level of detail
- **Built by Embedl:** Company focused on ultra-efficient SLMs (Small Language Models) for on-device AI
- Useful for understanding model internals without reading config files or code
