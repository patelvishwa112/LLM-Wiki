# World Models — Can Agents Learn Inside Their Own Dreams?

## Summary
A pointer to the influential "World Models" paper (worldmodels.github.io), which explores whether agents can learn inside their own internal simulations. This represents real reasoning and imagination, distinct from the simulated reasoning that LLMs perform.

## Key Takeaways

- **World Models:** Agents that build internal world representations and can "dream" — simulate experiences to learn without real-world interaction
- **Real reasoning vs. simulated reasoning:** World models involve actual internal simulation and imagination, not just pattern-matching on text as LLMs do
- This is a foundational concept for model-based reinforcement learning
